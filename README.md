# Proyectos de Ciberseguridad y Redes — Cisco Packet Tracer

Repositorio que recopila diferentes prácticas y proyectos realizados en Cisco Packet Tracer durante el Grado Superior de Desarrollo de Aplicaciones Multiplataforma (DAM), enfocados en redes, ciberseguridad, IoT y administración de infraestructuras.


# Proyectos incluidos

## VPN Site-to-Site
Implementación de una VPN IPsec entre varias sedes utilizando routers Cisco.

### Tecnologías y conceptos
- VPN IPsec
- ISAKMP
- AES-256
- ACLs
- Enrutamiento estático

### Contenido
- Configuración de routers
- Comunicación cifrada entre redes
- Verificación mediante pruebas de conectividad

Carpeta: `/vpn`


## Seguridad por MAC (Port Security)
Implementación de diferentes métodos de control de acceso mediante seguridad por direcciones MAC en switches Cisco.

### Tecnologías y conceptos
- Port Security
- Sticky MAC
- Protect
- Restrict
- Maximum MAC
- Seguridad en capa 2

### Contenido
- Configuración de puertos seguros
- Restricción de dispositivos no autorizados
- Verificación mediante monitorización del switch

 arpeta: `/mac-security`


## Configuración de Cortafuegos (ASA)
Configuración de un cortafuegos Cisco ASA para segmentación de red y control de tráfico entre zonas de seguridad.

### Tecnologías y conceptos
- Cisco ASA
- Security Levels
- DHCP
- Segmentación de red
- Seguridad perimetral

### Contenido
- Red inside / outside
- Configuración de interfaces
- Servicios de red básicos

Carpeta: `/firewall`


## IoT + AAA
Implementación de una arquitectura IoT con autenticación centralizada mediante AAA (RADIUS) y servicios DHCP.

### Tecnologías y conceptos
- AAA (RADIUS)
- DHCP
- IoT
- RFID
- Redes inalámbricas
- Control de acceso

### Contenido
- Gestión de dispositivos IoT
- Autenticación centralizada
- Automatización básica
- Control mediante RFID

Carpeta: `/iot-aaa`


# Tecnologías utilizadas

- Cisco Packet Tracer
- Cisco IOS
- Cisco ASA
- Routing & Switching
- VPN IPsec
- DHCP
- AAA / RADIUS
- Seguridad de red
- IoT


# Objetivo del repositorio

Este repositorio funciona como portfolio técnico y recopilación de prácticas relacionadas con:
- Redes
- Ciberseguridad
- Infraestructura
- Administración de sistemas y comunicaciones

Todos los proyectos han sido realizados en entorno simulado utilizando Cisco Packet Tracer.
